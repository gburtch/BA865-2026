This is a 3-layer neural network with shared weights.
Why shared weights matter: The same Wx, Wh is used at every time step — so gradients from all steps update the same parameters. This also means the network can handle sequences of any length without adding parameters.
Training & Vanishing Gradients
Backprop through an unrolled RNN is standard backprop through a deep network — but the layers are tied, and gradients must multiply by Wh at every time step.
How backpropagation through time works
1.  Loss depends on h₃ (or all hₜ, depending on the task).
2.  To compute ∂L/∂Wh, gradients flow backward through every time step.
3.  Each backward pass multiplies the gradient by Wh (approximately).
4.  After T steps: gradient ∝ WhT — the same matrix raised to a power.
λ < 1  ·  Our Wh has eigenvalue 0.5
0.5T → 0 as T grows
Signals from early inputs shrink to near-zero. The model cannot learn long-range dependencies — it effectively forgets everything more than ~10–20 steps back.
λ > 1  ·  Exploding gradients
λT → ∞ as T grows
Training becomes numerically unstable. Gradient clipping is a common workaround, but it doesn't fix the root cause.
The practical consequence: A vanilla RNN trained on long sequences will essentially ignore everything more than ~20 steps back. It can't learn that a subject 50 words ago constrains the verb today. This is why vanilla RNNs underperform on real NLP tasks.
LSTM — One Idea Fixes the Problem
LSTMs add gates — sigmoid outputs that decide how much of the past to preserve. Information can pass through time with much less shrinkage.
The forget gate
ft = σ( Wf xt + Uf ht−1 )
· σ (sigmoid) squashes output to (0, 1)
· ft ≈ 1  →  keep the past state
· ft ≈ 0  →  forget the past state
· The gate is learned — the model discovers when to remember and when to forget
The cell state — the "memory highway"
ct = ft ⊙ ct−1 + it ⊙ gt
ft ⊙ ct−1  — how much of the old cell state to keep
it ⊙ gt  — how much new information to write
When ft ≈ 1, ct ≈ ct−1 — the gradient flows backward almost unchanged. That's the fix: the cell state is a highway that bypasses the repeated multiplication by Wh.
The Unifying Symmetry
Dense layers, CNNs, and RNNs all do the same thing: apply a learned transformation, somewhere. RNNs apply it across time. Transformers remove the bottleneck.

What you can now say
①  An RNN is a dense layer applied across time, output looped back as input.
②  Weights are shared — Wx and Wh are the same at every step.
③  Unrolled, it's a deep network — training is standard backprop through time (BPTT).
④  Vanishing gradients arise because repeated multiplication by Wh shrinks signals.
⑤  LSTMs add gates so the cell state can pass through time without shrinkage.
⑥  Transformers remove the bottleneck entirely — attention looks directly at any past position.
